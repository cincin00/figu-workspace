쿠지 전용 온라인 라운지

피규 라운지에서 할 수 있는 일부터 바로 이해되는 홈.

피규 라운지는 지금 쿠지 응모 · 당첨 결과 확인 · 보관 · 배송 요청만 빠르게 처리하는 서비스입니다. 쇼핑몰처럼 제품을 탐색하는 대신, 사용자가 지금 해야 할 행동과 시스템 흐름을 첫 화면에서 즉시 이해하도록 홈 구조를 재정리했습니다.

3 step flow

서비스 흐름

  1. 01
    진행 중인 쿠지를 고른다

    시리즈, 남은 수량, 가격, 마감 상태를 보고 바로 참여할 쿠지를 선택합니다.

  2. 02
    결제 후 즉시 결과를 확인한다

    로딩 없이 당첨 등급과 획득 상품, 남은 보관 상태가 한 번에 노출됩니다.

  3. 03
    보관하거나 배송 요청한다

    여러 번 뽑은 상품은 마이 라운지에 모아두고 원하는 때에 합배송 요청이 가능합니다.

Why this home

쇼핑몰형 진열보다 행동 중심 안내

첫 방문자에게 “무엇을 살 수 있나”보다 “여기서 무엇을 할 수 있나”를 먼저 전달합니다.

Primary task

한 화면에서 참여 → 확인 → 배송까지

쿠지 서비스 핵심 여정만 홈에서 노출해 선택 피로를 줄이고 이탈을 낮춥니다.

Mobile first

모바일에서도 짧은 스크롤로 이해

상단 요약, 단계 설명, 현재 진행 중 쿠지를 순서대로 배치해 작은 화면에서도 직관적입니다.

이용 흐름

당첨 결과와 보관 상태 확인

응모 직후 결과를 확인하고, 획득한 경품이 어떻게 보관되는지 이해할 수 있어야 합니다.

  • 결제 완료 후 즉시 결과 노출
  • 당첨 등급 및 획득 상품 기록 유지
  • 마이 라운지에서 보관 수량 확인
운영 신뢰

어떻게 운영되는지 투명하게 이해

랜덤 서비스 특성상 신뢰 요소를 별도 섹션이 아니라 홈 상단 근처에 명확히 보여주는 것이 중요합니다.

  • 정품 보장 및 운영 정책 고정 노출
  • 남은 수량과 진행 상태 실시간 반영
  • 배송/합배송 정책 접근 경로 단순화
